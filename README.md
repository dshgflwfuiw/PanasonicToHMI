# 松下PLC变量转HMI工具

一个用于将松下Control FPWIN Pro 7导出CSV变量文件转换为HMI(威纶通)标签通信可用格式的工具。

## 功能特点

- 支持松下PLC变量CSV文件的读取和转换
- 自动生成HMI格式的变量文件
- 实时日志显示转换进度
- 支持错误提示和警告信息
- 界面简洁直观

## 系统要求

- Windows 操作系统
- .NET 6.0 或更高版本

## 使用说明

1. 运行程序
2. 点击"选择并转换文件"按钮
3. 选择松下PLC导出的变量CSV文件
4. 等待转换完成
5. 转换后的文件将保存在原文件同目录下

## 开发环境

- Visual Studio 2022
- C# WinForms
- .NET 6.0
